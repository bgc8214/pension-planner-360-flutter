# 스크린샷 가이드

앱 스토어 제출용 스크린샷 촬영 가이드입니다.

## 📸 촬영된 스크린샷

### 필수 스크린샷 (스토어 제출용)

1. **01_home.png** - 홈 화면 (입력 탭)
2. **02_onboarding.png** - 온보딩 첫 화면

### 추가로 필요한 스크린샷

3. **입력 화면** - 사용자가 값을 입력하는 화면
4. **결과 화면 - 세액공제** - 연간 예상 환급액 카드
5. **결과 화면 - 미래자산** - 미래 예상 자산 카드  
6. **결과 화면 - 연금수령** - 연금 수령 시뮬레이션 카드
7. **차트 화면** - 자산 변화 차트
8. **시나리오 목록** - 저장된 시나리오 목록
9. **다크모드** - 다크 테마 화면
10. **설정 화면** - 설정 메뉴

## 🎯 스크린샷 요구사항

### Google Play
- **크기**: 1080 x 1920 (세로)
- **형식**: PNG 또는 JPEG
- **최소**: 2장
- **최대**: 8장
- **권장**: 5-6장

### App Store
- **iPhone 6.7"**: 1290 x 2796
- **iPhone 6.5"**: 1284 x 2778  
- **형식**: PNG 또는 JPEG
- **최소**: 1장
- **최대**: 10장

## 🛠️ 스크린샷 촬영 방법

### 방법 1: ADB 명령어

```bash
# 현재 화면 캡처
adb -s emulator-5554 shell screencap -p /sdcard/screenshot.png

# 스크린샷 다운로드
adb -s emulator-5554 pull /sdcard/screenshot.png docs/screenshots/

# 임시 파일 삭제
adb -s emulator-5554 shell rm /sdcard/screenshot.png
```

### 방법 2: 에뮬레이터 UI

1. Android 에뮬레이터 사이드바
2. "Take screenshot" 버튼 클릭
3. 저장 위치 선택

### 방법 3: 실제 기기

1. 실제 Android/iOS 기기에서 앱 실행
2. 기기의 스크린샷 기능 사용
   - Android: 전원 + 볼륨 다운
   - iOS: 사이드 버튼 + 볼륨 업
